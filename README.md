# Home

*trtl-py is a Python Wrapper for the JSON-RPC API of TurtleCoin, which integrates with TurtleCoind and WalletAPI.*
  
## Installation

```sh
pip3 install trtl
```

## Requirements

* Python &gt;= 3.6 
* TurtleCoin version 0.22.0
