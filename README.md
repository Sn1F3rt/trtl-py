<p align="center"><i>trtl-py is a Python Wrapper for the JSON-RPC API of TurtleCoin, which integrates with TurtleCoind and WalletAPI.</i></p>

  
## Installation

```sh
pip3 install trtl
```

## Requirements

* Python &gt;= 3.6 
* TurtleCoin version 0.24.0
